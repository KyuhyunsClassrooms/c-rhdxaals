# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
(프로젝트의 전체 작동 로직을 의사 코드로 작성하세요.)

시작
    반복 (사용자가 0번 '종료'를 선택할 때까지):
        메뉴 출력:
            1. 상품 등록
            2. 상품 목록 출력
            3. 상품 검색
            4. 총 재고 금액 계산
            0. 종료

        사용자에게 메뉴 번호 입력받기

        만약 선택이 1이면:
            상품 등록 함수 호출

        만약 선택이 2이면:
            상품 목록 출력 함수 호출

        만약 선택이 3이면:
            상품 검색 함수 호출

        만약 선택이 4이면:
            총 재고 금액 계산 함수 호출

        만약 선택이 0이면:
            반복 종료

    반복 끝
종료
