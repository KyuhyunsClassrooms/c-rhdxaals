> 💡 **디버깅이란?** 프로그램에서 발생한 오류(버그)를 찾아내고 고치는 과정입니다.

## 🐛 발견한 오류 #1

### 1) 어떤 문제가 발생했나요?
(예: 프로그램을 실행하면 이상한 값이 나왔어요 / 컴파일이 안 돼요)

프로그램을 실행하면 메뉴를 입력했을 때 엔터 입력 때문에 무한 반복되는 문제가 발생했습니다.
예를 들어 숫자를 입력했는데 바로 다음 메뉴 입력이 넘어가거나
입력이 정상적으로 처리되지 않는 문제가 생겼습니다.

### 2) 오류가 발생한 코드
scanf("%d", &menu);


### 3) 왜 오류가 생겼나요?
(예: 세미콜론(;)을 빼먹었어요 / 변수 이름을 잘못 썼어요)

scanf("%d") 사용 시 버퍼에 남아 있는 엔터(\n) 문자 때문에
다음 입력을 받을 때 문제가 생길 수 있기 때문입니다.
### 4) 어떻게 고쳤나요?
scanf("%d", &menu);
while(getchar() != '\n');   // 입력 버퍼 비우기 추가


### 5) 고친 후 결과
- [ ] ✅ 잘 작동해요!
- [o] ❌ 아직 문제가 있어요

## 🐛 발견한 오류 #2

### 1) 어떤 문제가 발생했나요?
코드 자체적으로 문제가 많았음

### 2) 오류가 발생한 코드
전부

### 3) 왜 오류가 생겼나요?
ai를 활용할 때 구체적으로 말하지 않아서

### 4) 어떻게 고쳤나요?
이미 작성해둔 알고리즘을 통해서 더욱 구체적으로 ai에게 요구하여 코드를 작성함
### 5) 고친 후 결과
- [o] ✅ 잘 작동해요!
- [ ] ❌ 아직 문제가 있어요


## 🧪 프로그램 테스트 결과

### 테스트 1
- **무엇을 테스트했나요?** 상품등록 상품목록확인 재고검색 총 금액 계산
- **어떻게 테스트했나요?** 상품에 사과를 적고 직접 금액과 재고 개수를 입력했습니다
- **결과:** ✅ 성공 

### 테스트 2
- **무엇을 테스트했나요?** 여러가지 상품을 등록하고 총 금액 계산을 했습니다
- **어떻게 테스트했나요?** 직접 작성한 코드에서 4 번을 눌러서 테스트 했습니다
- **결과:** ✅ 성공

## 💭 디버깅하면서 배운 점

이번 프로젝트에서 오류를 고치면서 새롭게 알게 된 점이나 느낀 점을 자유롭게 적어보세요.

이번 활동을 하면서 인공지능도 잘 활용하려면 나도 코드에 대해서 잘 알아야만 하고
또, 인공지능을 잘 활용할 수 있는 사람이 더 성장할 수 있겠다는 것을 깨달았습니다.
